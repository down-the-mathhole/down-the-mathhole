Sketch 1: Animated Sine Wave

This sketch demonstrates a basic animated sine wave. The sine function is one of the most fundamental functions in mathematics, with applications ranging from sound waves to alternating current.

The sine function is defined as: \[ y = A \sin(kx - \omega t) \] Where:

  • \(A\) is the amplitude (height) of the wave
  • \(k\) is the wave number (related to the wavelength)
  • \(\omega\) is the angular frequency
  • \(t\) is time

In the animation above, we're using p5.js to create a moving sine wave. The wave is drawn using a series of ellipses, each positioned according to the sine function at a given point in time.

The movement of the wave is achieved by incrementing the phase (\(\theta\)) in each frame: \[ \theta_{new} = \theta_{old} + \Delta\theta \] Where \(\Delta\theta\) determines the speed of the wave's motion.
